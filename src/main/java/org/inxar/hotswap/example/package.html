This package provides code used to run the examples. The main class is RunExample (paths are relative to the main installation directory):

Examples Usage Summary 

# Path where JDK1.2 or JDK1.3 is installed
JAVA_HOME=/usr/java/jdk1.3

# Path where HotSwap is installed
HOTSWAP=/usr/share/java/inxar/hotswap-x_y_z

# Name of JDK1.2 or JDK1.3 interpreter
java \

 # Classpath for the main application
 -classpath $HOTSWAP/hotswap.jar:$JAVA_HOME/lib/tools.jar \

 # Main class
 example.RunExample \

 # Path where classfiles are written
 $HOTSWAP/examples \

 # Path where sourcefiles are searched
 $HOTSWAP/examples \

 # Classpath for the compilation command
 "$HOTSWAP/hotswap.jar" \

 # Name of the class to be hotswapped (located in $HOTSWAP/examples)
 HelloWorld

You can substitute HelloWorld for an Example implementation of your choice. Although shown here, the tools.jar is optional. There is a helloworld.sh script in the ./bin directory that runs exactly what is shown above. I've also written a helloworld.bat file, but I have my doubts about its operability.
